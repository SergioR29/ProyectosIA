# ProyectosIA
Este es un repositorio en el que doy a conocer proyectos personales de desarrollo de software que he creado mediante el empleo de metodologías VibeCoding + IA:

- [LinkDirectory](https://github.com/SergioR29/LinkDirectory): Página web generada mediante el empleo de la ingeniería de prompts con Antigravity en JS y Node.js. Sirve para guardar enlaces de webs que nos parezcan importantes y/o interesantes y son agrupables mediante diferentes categorías. Y desde ahí puedes hacer clic a cualquiera de los enlaces guardados para visitarlos.
